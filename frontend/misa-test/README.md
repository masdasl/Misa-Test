# Project Name

## Mô tả dự án

Dự án này là một ứng dụng web quản lý khách được xây dựng bằng **Vue 3**, **TypeScript** và **Vite**.

## Hướng dẫn phát triển (Dev)

### 1. Cài đặt dependencies

npm install

# hoặc

yarn install

### 2. Tạo file .env

VITE_API_PROTOCOL= # Giao thức backend (http hoặc https)
VITE_API_HOST= # Địa chỉ host backend
VITE_API_PORT= # Cổng backend

### 3. Chạy dự án cho dev

npm run dev

# hoặc

yarn dev
